# Resourceful Sheep Mod for Minecraft (Under Development)

Resourceful Sheep is an upcoming Minecraft mod that aims to introduce a new gameplay mechanic surrounding sheep. Once completed, sheep in this mod will drop special Wools when sheared. These special Wools can then be processed to obtain various materials.

## Planned Features

- Discover various types of sheep each yielding a unique type of Wool.
- Shear the sheep to collect special Wools.
- Process the Wools to obtain different materials.
- Explore, Collect, and Craft!

## Project Status

This project is currently in the initial development stage. The described features have not been implemented yet.
